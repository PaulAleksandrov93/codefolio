# Codefolio

Codefolio - это веб-приложение для создания портфолио проектов. Он разработан с использованием Django REST Framework и ReactJS.

## Особенности

- Добавление проектов с описаниями, изображениями и ссылками.
- Связывание технологий с проектами.
- Редактирование и удаление проектов.
- Просмотр списка проектов и подробной информации о каждом проекте.

## Использование

Этот проект предназначен для демонстрации проектов в области разработки.
